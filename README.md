# picoCTF Writeups

Repository này được tạo ra để ghi lại quá trình mình giải các bài CTF trên nền tảng picoCTF.

## Mục tiêu

- Lưu lại cách tư duy và các bước giải từng challenge.
- Tổng hợp kiến thức theo từng mảng (Crypto, Web, Forensics, Binary Exploitation, ...).
- Dùng như tài liệu ôn tập cho các mùa picoCTF tiếp theo.

## Cấu trúc repository

Các writeup được chia theo nhóm challenge:

- Binary Exploitation/
- Blockchain/
- Cryptography/
- Forensics/
- General Skills/
- Reverse Engineering/
- Web Exploitation/

Mỗi bài thường có:

- File markdown ghi lại quá trình giải.
- File đính kèm gốc của đề (nếu có): binary, script, ciphertext, ...

## Cách mình ghi writeup

Trong mỗi writeup, mình cố gắng giữ format nhất quán:

1. Tóm tắt đề bài.
2. Quan sát ban đầu.
3. Hướng giải và reasoning.
4. Script/command đã dùng.
5. Kết quả và flag.
6. Bài học rút ra.

## Lưu ý

- Repo này phục vụ mục đích học tập và tự tổng hợp kiến thức.
- Không đăng flag của các challenge đang active nếu điều đó vi phạm quy định cuộc thi.

## Tiến độ

Mình sẽ cập nhật dần writeup theo thời gian trong quá trình học và luyện CTF.
